# shop 패키지
- shop 을 관리하는 패키지


---
###  `ItemType.java`
- 아이템 종류를 구분하는 enum
  - `FOOD`: 경험치로 구매 → 체력 회복
  - `CLOTH`: 경험치로 구매 → 의류 착용 효과

### `ItemData.java`
- 고정된 아이템들을 생성하여 List로 제공
- UI에서 버튼 생성 시 활용

###  `ShopEffect.java`
- 아이템 구매 시, 플레이어 상태 변화 처리 담당
- 조건: 경험치가 충분해야만 구매 가능
- 효과: 체력 증가, 메시지 출력 등
